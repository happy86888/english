模型 建議用 gemini-2.0-flash（速度快、免費額度大）

YouTube Data API key

影片頁的「🔥 每日新鮮」分類會用這把 key 從 YouTube 抓新影片。 不填也能用其他功能，只是「每日新鮮」會顯示提示。

申請步驟： Google Cloud Console → 建立專案 → 啟用「YouTube Data API v3」 → 建立 API 金鑰。

搜尋程度 決定「每日新鮮」抓哪種難度的影片
各程度搜尋關鍵字 可以調整成你偏好的主題（例如加上 "cooking"、"tech"、"news"）。修改後重新點「每日新鮮」即可生效。
